# Configuración de Minikube

## Instalar Minikube (si no está instalado)
winget install Kubernetes.minikube

## Iniciar Minikube
minikube start

## Habilitar metrics-server para el autoscaler (HPA)
minikube addons enable metrics-server


cd back/k8s
kubectl apply -f .

cd front/k8s
kubectl apply -f .
